Numbers and Operators

In JavaScript, you can also perform basic mathematical operations like Addition, Subtraction, Multiplication, and Division.

Examples

Addition

400 + 20;

You can also add multiple numbers

22 + 33 + 44;

JavaScript can also do subtraction...

1000 - 17;

Perform multiplication, using an asterisk(*)... 

123 * 456;

Division, using forward slash

12345 / 250;

You can also combine these simple operations to make something more complex, like this:

1234 + 57 * 3 - 31 / 4;

What if you want to do addition and subtraction first, before doing multiplication and division?

Say you have 1 brother, 3 sisters and 8 candies. You want to split the candies equally among your 4 siblings. (You have already taken your share.) You have to divide 8 by your number of siblings. 

8 / 1 + 3;

This is not right!

8 / (1 + 3);

Now that is better. You provide 2 candies to each your siblings.

Take note that the parentheses() forces JavaScript to add 1 and 3 before dividing 8 by 4.

Variables

Javascript lets you give names to values using variables.

To create a variable, we use the keyword var

JavaScript knows that we are about to enter the name of a new variable.

var pogi;

Review

Strings and numbers alone return a value.

1 + 2 + 3 + 4 * 5 / 20 - 7; 0

Creating a variable in JavaScript does not return a value.

var forever; undefined

Note: JavaScript will output undefined whenever a command doesn't return a value.

Giving a value to a variable

var age = 20;
undefined

Note: Setting a value is called assignment.

age;
20

Remember

Variables can vary.

If we want to update it, we just use = again.

age = 13;
13

Note: We need to use var only when we want to create a variable.

The Candy Problem (Revisited)

var numberOfSiblings = 1 + 3;
var numberOfCandies = 8;
numberOfCandies / numberOfSiblings;
2
Naming Variables

Note: Be careful with naming variables.

  • Easy to misspell
  • Capitalization

Let's say, we accidentally used a lowercase c in numberOfCandies

numberOfcandies / numberOfSiblings;
ReferenceError: numberOfcandies is not defined

The Tricky Parts

Unfortunately, JavaScript will only do exactly what you ask it to do.

Variable names in JavaScript CANNOT contain spaces.

number of candies
numberofcandies

The Camelcase

One common way to get around this is to start each word with a capital letter:

numberOfCandies

In standard practice, it is common for variables to start with a lowercase letter.

After that, capitalize each word except for the first one

Strings

Strings in JavaScript (as in most programming languages) are just sequences of characters, which can include:

  • Letters
  • Numbers
  • Punctuations
  • Spaces

We put strings between quotes("") so JavaScript knows where they start and end.

"Hello, Crush!";
Hello, Crush!

We can also put strings in between single quotes('')

'...';
...

We can also save strings into variables, just like numbers:

var myName = "Papa J";

We can also assign a string to a variable that previously contained a number or a string:

var something = 5;
something = "special";
"special"

What if we put a number between quotes? Is it a string or a number?

var numberNine = 9;
var stringNine = "9";
numberNine + numberNine;
18
stringNine + stringNine;
"99"

When we add the number values of two nines(9), we get 18.

However, when we use the operator "+" on two strings of nines(9), the strings are simply joined together to form "99"

What if we add a string and a number?

stringNine + numberNine;
"99"

Converting a value from one data type to another.

This happens quite often in JavaScript because it's dynamically typed.

Joining Strings

We can use the "+" operator with strings, but the result is very different from using the "+" with numbers.

When we use "+" to join two strings, we make a new string with the second string attached to the end of the first string:

var greeting = "Hello";
var myName = "Sef";
greeting + myName;
"HelloSef"

Note: Javascript does not automatically put a space unless we specifically tell it to by adding a space in one of the original strings:

var greeting = "Hello ";
var myName = "Sef";
greeting + myName;
"Hello Sef" var greeting = "Hello";
var myName = "Sef";
greeting + " " + myName;
"Hello Sef"

Arrays

An array is a list of other JavaScript data values.

JavaScript lets us create and group together data in more interesting ways with arrays

Say, you and your crush are talking about your top three(3) favorite foods.

You could create an array consisiting your top favorite foods, in order:

var myTopThreeFavoriteFoods = ["Pizza", "Sinigang", "Kare-Kare"];

So, instead of giving your crush three seperate strings, you can just us the single array myTopThreeFavoriteFoods.

Why should we care about arrays?

Let's look at a different example, Pokemons.

Say you want to use a program to keep track of the many kinds of Pokemons you know about. You could create a variable for each pokemons, like this:

Creating an Array

To create an array, we just use square brackets, [].

In fact, an empty array is simply a pair of square brackets, like this:

[];
[]

Accessing an array's elements

When it's time to access elements in an array, we use square brackets with the index of the element we want:

pokemons[0];
"Bulbasaur"
pokemons[2];
"Charmander"

Note: An index is the number that corresponds to the spot in the array where a value is stored.

Note: The first element in an array is at index 0, the second is at index 1, the third at index 2, ans so on.

pokemons[0];
"Bulbasaur"
pokemons[2];
"Charmander"
